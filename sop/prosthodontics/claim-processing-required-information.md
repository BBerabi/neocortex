To process a prosthodontic treatment claim (bridges and removable dentures), the following information is necessary:

## Mandatory Information
*   **Teeth Replaced**: Identify the specific teeth being replaced.
*   **Type of Prosthesis**: Specify the type of prosthesis (e.g., fixed bridge, complete denture, partial denture).
*   **Bridge Units**: For a bridge, state the number and position of the units (artificial teeth and supporting crowns).

## Additional Documentation Requirements
*   **Clinical Notes**: Clinical notes are required.
*   **Radiograph**: A radiograph showing the edentulous (toothless) area is required.
*   **For Teeth Lost Before Cover**: If any tooth was lost before the member's cover began, the member must provide documentation establishing the date of loss so that the 'missing-tooth provision' can be assessed.
*   **For Replacement Prosthesis**: For a replacement prosthesis, the documentation must state the age of the existing prosthesis and the reason it can no longer serve.
